# Project LaIN

一个支持长期记忆, 具有表情切换, 语音识别以及语音输出的AI Waifu.

![image.png|900](https://s2.loli.net/2025/01/10/bTomcAwVfGUj9se.png)

## 功能

- 拥有长期记忆
- 立绘动态显示
- 语音识别输入
- 语音合成输出
- 声纹识别
  - 可配置限定用户群体语音识别
  - 说话人身份识别功能 (ps: 可配置模型prompt登记主人身份)
- 多种LLM支持
